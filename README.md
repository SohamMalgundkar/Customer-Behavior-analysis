# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, covering data loading, exploration, cleaning, SQL analysis, dashboard creation, and reporting. The goal is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

The dataset used for this project contains business-related information and was analyzed to identify trends, patterns, and key performance indicators (KPIs).

**Dataset Features:**

* Customer information
* Transaction details
* Product-related data
* Performance metrics

## Project Workflow

### 1. Data Loading

* Imported dataset into Python.
* Verified data structure, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Identified trends, distributions, and correlations.
* Visualized key patterns using charts and graphs.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Prepared data for analysis.

### 4. SQL Analysis

* Imported cleaned data into PostgreSQL/MySQL/SQL Server.
* Executed SQL queries for:

  * Data retrieval
  * Aggregations
  * Filtering
  * Joins
  * KPI calculations
  * Business insights generation

### 5. Dashboard Development

* Connected Power BI to the processed dataset.
* Created interactive dashboards and visual reports.
* Added filters, slicers, and KPI cards for better analysis.

### 6. Reporting

* Documented findings and business insights.
* Summarized key observations and recommendations.

### 7. Presentation

* Designed a professional project presentation using Gamma AI.
* Included project objectives, methodology, dashboard screenshots, and conclusions.

## How to Run

### Step 1: Clone the Repository

```bash
git clone <repository-link>
cd Data-Analytics-Project
```

### Step 2: Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy
```

### Step 3: Run Python Analysis

```bash
python data_loading.py
python eda.py
python data_cleaning.py
```

### Step 4: Execute SQL Queries

* Import the dataset into PostgreSQL, MySQL, or SQL Server.
* Run the SQL scripts provided in the `SQL Queries` folder.

### Step 5: Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop.
* Refresh the data connection if required.
